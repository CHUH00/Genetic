## 📌 Project Overview
This repository contains a Python implementation of **Genetic Algorithms (GA)** inside a Jupyter Notebook environment (`genetic.ipynb`). Genetic Algorithms are search heuristics inspired by Charles Darwin's theory of natural evolution, reflecting the process of natural selection where the fittest individuals are selected for reproduction.

This project is aimed at understanding the core mechanics of evolutionary computing and applying those principles to complex optimization or search problems. 

## ✨ Key Features
- **Initialization**: Generation of an initial population pool of potential solutions.
- **Fitness Evaluation**: Calculating the performance (fitness) of each individual against the target objective.
- **Selection Mechanism**: Identifying the best-performing individuals to become parents of the next generation (e.g., Roulette Wheel or Tournament selection).
- **Crossover & Mutation**: Executing bio-inspired operators to produce diverse offspring and maintain genetic variance, avoiding local optima.
- **Convergence Tracking**: Iterative generation cycles until a satisfactory solution or performance convergence is reached.

## 🛠️ Technology Stack
* **Language**: `Python 3.x`
* **Environment**: `Jupyter Notebook`
* **Core Libraries**: `NumPy`

## 🚀 Getting Started

### Prerequisites
Make sure you have Python and Jupyter installed:
```bash
pip install numpy jupyter
```

### Running the Notebook
1. Clone the repository:
   ```bash
   git clone https://github.com/CHUH00/Genetic.git
   cd Genetic
   ```
2. Launch Jupyter Notebook:
   ```bash
   jupyter notebook
   ```
3. Open `genetic.ipynb` and step through the algorithm's lifecycle.
